# 大学生创新创业大赛后端技术栈分析

基于第十六届大学生创新创业大赛A类企业赛题的后端技术选型指南。

---

## 一、AI/大模型类（占比最大）

**涉及题目**：A01-A05, A07-A11, A14, A15, A18, A19, A21, A24, A25

| 组件 | 推荐方案 | 说明 |
|------|----------|------|
| **语言** | Python | AI生态最成熟 |
| **框架** | FastAPI / Flask | 异步高性能，适合模型推理服务 |
| **AI框架** | PyTorch / PaddlePaddle（百度题用飞桨） | 模型训练与推理 |
| **大模型接入** | 文心API / OpenAI API / 本地部署LLM | 视题目要求 |
| **向量数据库** | Milvus / Chroma（RAG场景如A15） | 知识检索增强 |
| **数据库** | PostgreSQL + Redis | 结构化存储+缓存 |
| **消息队列** | RabbitMQ / Celery | 异步任务处理（模型推理耗时） |

---

## 二、IoT/智能家居类

**涉及题目**：A06（OpenHarmony智能家居）

| 组件 | 推荐方案 | 说明 |
|------|----------|------|
| **语言** | Python / Go / Java | Go性能好，Java生态成熟 |
| **框架** | Spring Boot（Java）或 FastAPI | 设备管理后端 |
| **IoT协议** | MQTT（EMQ X / Mosquitto） | 设备通信核心 |
| **数据库** | InfluxDB（时序）+ MySQL | 设备状态、历史数据 |
| **边缘计算** | OpenHarmony设备端 | 本地控制逻辑 |
| **消息** | Kafka / EMQX | 高吞吐设备消息 |

---

## 三、视觉检测/图像处理类

**涉及题目**：A07（眼底诊断）、A08（缺陷检测）、A10（车辆检测）、A25（浓烟人体识别）

| 组件 | 推荐方案 | 说明 |
|------|----------|------|
| **语言** | Python | CV生态 |
| **框架** | FastAPI + Celery | 图像推理异步处理 |
| **CV框架** | OpenCV + YOLO / MMDetection | 目标检测 |
| **存储** | MinIO / OSS | 图像文件存储 |
| **GPU推理** | TensorRT / ONNX Runtime | 加速部署 |

---

## 四、区块链/安全类

**涉及题目**：A13（分布式数字身份）、A17（安全漏洞实验场）

| 组件 | 推荐方案 | 说明 |
|------|----------|------|
| **语言** | Go / Java / Rust | 性能与安全 |
| **区块链** | Hyperledger Fabric / FISCO-BCOS | 国产联盟链 |
| **安全** | Docker隔离 + 沙箱环境 | 漏洞靶场 |
| **框架** | Spring Boot / Gin | 管理后端 |

---

## 五、元宇宙/3D类

**涉及题目**：A16（文博元宇宙）、A23（虚拟演播室）

| 组件 | 推荐方案 | 说明 |
|------|----------|------|
| **语言** | Node.js / Go | 实时性要求高 |
| **实时通信** | WebSocket / WebRTC | 多人同步 |
| **3D资产** | 对象存储（OSS/MinIO） | 模型文件管理 |
| **引擎后端** | Unreal Pixel Streaming | 云渲染（A23） |

## 六、工业MES类

**涉及题目**：A12（低代码MES）

| 组件 | 推荐方案 | 说明 |
|------|----------|------|
| **语言** | Java | 企业级首选 |
| **框架** | Spring Boot + Spring Cloud | 微服务架构 |

## 七、新手学习路径建议

### 通用基础（所有方向必学）

1. **Python 基础** — 语法、面向对象、常用库
2. **HTTP/REST API 概念** — 请求响应、状态码、JSON
3. **数据库基础** — SQL语法、PostgreSQL或MySQL
4. **Git 版本控制** — 基本操作、分支管理
5. **Docker 容器** — 镜像构建、容器运行、docker-compose

### 按方向深入

| 方向 | 核心技能 | 学习周期 |
|------|----------|----------|
| AI/大模型 | FastAPI + PyTorch + 大模型API | 2-3个月 |
| IoT智能家居 | MQTT协议 + 时序数据库 | 1-2个月 |
| 视觉检测 | OpenCV + YOLO + 图像处理 | 2-3个月 |
| 区块链/安全 | Go/Rust + 密码学基础 | 3-4个月 |
| 元宇宙 | WebSocket + 3D基础 | 2-3个月 |
| 工业MES | Spring Boot + 微服务 | 3-4个月 |
